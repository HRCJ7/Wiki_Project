# Wiki-Project
This project is designed for the User who is exploring the city and looking for nearby places to visit.
## Project Infomation 

### Project Architecture:

 MVVM architecture 

### Main Libraries which are used in project:

1. Retrofit
2. Dagger2
3. RxJava
4. Glide
5. Google Services
